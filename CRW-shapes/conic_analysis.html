Testing Hyperbola and ellipse functions¶

Parametric equations for conics: \begin{align} x = a C(t) - x_0 \ y = b S (t) \end{align}

Where: \begin{align} C(t) = \cos t ~\mathrm{if~ ellipse} \ C(t) = \cosh t ~\mathrm{if~ hyperbola} \ S(t) = \sin t ~\mathrm{if~ ellipse} \ S(t) = \sinh t ~\mathrm{if~ hyperbola}\ R_c = \frac{b^2}{a}\ \tan\theta_c = \frac{b}{a}\ a = \frac{R_c}{\tan^2\theta_c} \ b = \frac{R_c}{\tan\theta_c} \end{align} $a$ takes (-) sign for hyperbolae

$R_{90}$ for ellipse¶

$t_{90}$ is such that: \begin{align} x{90} = a-R_0 = a\cos t{90} \ y{90} = R{90} = b\sin t{90} \ R{90} = b\left(1-\frac{(R_0-a)^2}{a^2}\right) \end{align}

$R_{90}$ for hyperbola¶

in this case: \begin{align} x{90} = a\cosh t{90} = a + R0 \ y{90} = R{90} = b\sinh t{90} \ R_{90} = b\left(1+\frac{(R_0 + a)^2}{a^2}\right) \end{align}

$R_{90}$ general¶

$R_{90} = b\left(1 \mp \frac{(R_0 - a)^2}{a^2}\right)$

(-) for ellipse (+) for hyperbola
