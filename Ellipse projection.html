So, with $\theta_{\mathrm{c}} = 25^\circ$ (prolate ellipsoid), then for $\theta' < 60^\circ$ and $i < 60^\circ$, we need $\theta$ up to $110^\circ$.

If we wanted to go to $i = 75^\circ$ then we would need up to $\theta = 130^\circ$.

Projected shape factors: $A'$ versus $q'/q$ for conic sections¶

These are the results for spheroids that I derived in my handwritten notes. $A = R_ {\mathrm{c}} / R_ 0$ and $A'$ is the projected version of the same. $q = R_ 0/ D$ and $q'$ is the projected version of the same.

The dots are shown every $15^\circ$ up to $75^\circ$. There is a limiting value of $A'$ for each value of $\theta_ {\mathrm{c}}$, which is the limit as $i \to 90^\circ$. This is 1 for circles $\theta_ {\mathrm{c}} = 45^\circ$ and 2 for parabolas $\theta_ {\mathrm{c}} = 0$.
